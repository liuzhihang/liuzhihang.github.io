ES 安装 search-guard-6

安装插件

☞ 官方网站
在ES目录下执行命令

bin/elasticsearch-plugin install -b com.floragunn:search-guard-6:6.4.2-23.1

注: 安装版本需要和Elasticsearch版本相对应. 查看版本

这里不使用官方的快速构建方法
