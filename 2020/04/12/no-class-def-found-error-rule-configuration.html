问题定位

  1. 开始排查是因为缺少 sharding-core-api-4.0.0-RC2.2-1.8.jar 包下的一个文件, 但是本地是有的
  2. 本地可以启动
  3. 服务器启动失败, 可能是jar包缺少
  4. 最后结果发现 sharding-core-api-4.0.0-RC2.2-1.8.jar 是通过公司封装的一个包传递进来的, 而封装的那个jar包在私服上已经被删除了. 删除原因
